2. Describe rooms and furniture arrangements
3. Ask about prices and negotiate when shopping
4. Use prepositions to describe item locations

## 2. Essential Verbs and Actions

| Infinitive | Present (ich) | English | Example |
|------------|---------------|---------|----------|
| stellen | stelle | to place (vertical) | Ich stelle die Lampe auf den Tisch. |
| legen | lege | to place (horizontal) | Ich lege das Buch auf den Tisch. |
| hängen | hänge | to hang | Ich hänge das Bild an die Wand. |
| sitzen | sitze | to sit | Ich sitze auf dem Sofa. |
| liegen | liege | to lie | Das Kissen liegt auf dem Bett. |
| kosten | kostet | to cost | Der Tisch kostet 100 Euro. |
| einrichten | richte ein | to furnish | Ich richte meine Wohnung ein. |
| aufbauen | baue auf | to assemble | Ich baue den Schrank auf. |

## 3. Prepositions and Location

When describing furniture placement, use these prepositions with the correct case:

### Dative (location - where something is)
- **auf** dem Tisch (on the table)
- **in** dem/im Zimmer (in the room)
- **neben** dem Bett (next to the bed)
- **zwischen** den Stühlen (between the chairs)
- **unter** dem Tisch (under the table)

### Accusative (direction - where something is being moved to)
- **auf** den Tisch (onto the table)
- **in** das/ins Zimmer (into the room)
- **neben** das Bett (next to the bed)
- **zwischen** die Stühle (between the chairs)
- **unter** den Tisch (under the table)

:::tip Grammar Note
Use accusative case when describing movement/direction (Wohin? - Where to?) and dative for location (Wo? - Where?).
:::

## 6. Interactive Exercises 🎮

### Exercise 1: Article Practice

export const ArticleQuiz = () => {
  const [answers, setAnswers] = React.useState({});
  const [showResults, setShowResults] = React.useState(false);
  const [score, setScore] = React.useState(0);
  
  const questions = [
    { word: 'Tisch', correct: 'der', icon: '🪑' },
    { word: 'Lampe', correct: 'die', icon: '💡' },
    { word: 'Bett', correct: 'das', icon: '🛏️' },
    { word: 'Stuhl', correct: 'der', icon: '🪑' },
    { word: 'Sofa', correct: 'das', icon: '🛋️' }
  ];

  return (
    <div className="quiz-container">
      <div className="quiz-header">
        <h4>Choose the correct article</h4>
        {showResults && <div className="quiz-score">Score: {score}/{questions.length}</div>}
      </div>
      <div className="quiz-items">
        {questions.map((q, idx) => (
          <div key={idx} className="quiz-item">
            <div className="quiz-word">
              <span className="quiz-icon">{q.icon}</span>
              <span className="word">{q.word}</span>
            </div>
            <div className="article-buttons">
              {['der', 'die', 'das'].map(article => (
                <button
                  key={article}
                  onClick={() => setAnswers({...answers, [idx]: article})}
                  disabled={showResults}
                  className={`article-button ${
                    answers[idx] === article ? 'selected' : ''
                  } ${
                    showResults ? 
                      article === q.correct ? 'correct' : 
                      answers[idx] === article ? 'incorrect' : '' 
                    : ''
                  }`}
                >
                  {article}
                </button>
              ))}
            </div>
          </div>
        ))}
      </div>
      <div className="exercise-buttons">
        <button 
          onClick={() => {
            setShowResults(true);
            setScore(questions.filter((q, idx) => answers[idx] === q.correct).length);
          }}
          className="check-button"
        >
          Check Answers
        </button>
        <button 
          onClick={() => {
            setAnswers({});
            setShowResults(false);
            setScore(0);
          }}
          className="reset-button"
        >
          Reset
        </button>
      </div>
    </div>
  );
};

### Exercise 2: Price Learning Game 💶

export const PricePractice = () => {
  const [currentStep, setCurrentStep] = React.useState(0);
  const [answer, setAnswer] = React.useState('');
  const [showHint, setShowHint] = React.useState(false);
  const [showResult, setShowResult] = React.useState(false);

  const prices = [
    { 
      price: "€149,99",
      display: "💶 149,99",
      answer: "einhundertneunundvierzig Euro neunundneunzig Cent",
      hint: "100 = einhundert, 40 = vierzig, 9 = neun" 
    },
    { 
      price: "€299,-",
      display: "💶 299,00",
      answer: "zweihundertneunundneunzig Euro",
      hint: "200 = zweihundert, 90 = neunzig, 9 = neun" 
    },
    { 
      price: "€75,50",
      display: "💶 75,50",
      answer: "fünfundsiebzig Euro fünfzig Cent",
      hint: "70 = siebzig, 5 = fünf, 50 = fünfzig" 
    },
    { 
      price: "€25,99",
      display: "💶 25,99",
      answer: "fünfundzwanzig Euro neunundneunzig Cent",
      hint: "20 = zwanzig, 5 = fünf, 99 = neunundneunzig" 
    }
  ];

  const handleNext = () => {
    if (currentStep < prices.length - 1) {
      setCurrentStep(curr => curr + 1);
      resetState();
    }
  };

  const handlePrevious = () => {
    if (currentStep > 0) {
      setCurrentStep(curr => curr - 1);
      resetState();
    }
  };

  const resetState = () => {
    setAnswer('');
    setShowHint(false);
    setShowResult(false);
  };

  const checkAnswer = () => {
    setShowResult(true);
  };

  return (
    <div className="price-game">
      <div className="price-card">
        <div className="price-progress">
          Question {currentStep + 1} of {prices.length}
        </div>
        <div className="price-display">{prices[currentStep].display}</div>
        <div className="price-input-area">
          <input
            type="text"
            value={answer}
            onChange={(e) => setAnswer(e.target.value)}
            placeholder="Write the price in German..."
            className="price-input"
            disabled={showResult}
          />
          <button 
            className="hint-button"
            onClick={() => setShowHint(!showHint)}
          >
            💡 {showHint ? 'Hide' : 'Show'} Hint
          </button>
        </div>
        {showHint && (
          <div className="hint-text">
            <span>🔍 Hint: {prices[currentStep].hint}</span>
          </div>
        )}
        {showResult && (
          <div className={`result-message ${
            answer.toLowerCase() === prices[currentStep].answer.toLowerCase() 
              ? 'correct' 
              : 'incorrect'
          }`}>
            {answer.toLowerCase() === prices[currentStep].answer.toLowerCase() 
              ? '✅ Correct!' 
              : `❌ The correct answer is: ${prices[currentStep].answer}`}
          </div>
        )}
        <div className="button-group">
          <button 
            className="check-button"
            onClick={checkAnswer}
            disabled={!answer || showResult}
          >
            Check Answer
          </button>
          <button 
            className="reset-button"
            onClick={resetState}
          >
            Reset
          </button>
        </div>
        <div className="navigation-buttons">
          <button 
            className="nav-button"
            disabled={currentStep === 0}
            onClick={handlePrevious}
          >
            ← Previous
          </button>
          <button
            className="nav-button"
            disabled={currentStep === prices.length - 1}
            onClick={handleNext}
          >
            Next →
          </button>
        </div>
      </div>
    </div>
  );
};

## Cultural Note

Germans often use specific measurements when discussing furniture and room sizes. Measurements are typically in meters and centimeters, and prices are in Euros and Cents. Being precise with measurements and prices is an important part of German furniture shopping culture.
